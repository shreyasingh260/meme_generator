# 🎭 Meme Machine 3000

A **zero-dependency, no-watermark** meme generator that runs entirely in the browser. No backend, no sign-up, just pure meme energy.

## ✨ Features

- 📁 **Upload your own image** (PNG, JPG, GIF, WEBP)
- 🔥 **6 classic meme templates** (Drake, Distracted BF, Two Buttons, and more)
- ✏️ **Top & Bottom text** with full style controls
- 🎨 **Font, size, color, stroke** customization
- 😂 **16 drag-and-drop emoji stickers**
- 🖱️ **Draggable text & sticker layers** on canvas
- ⬇️ **Download as PNG** — no watermark, ever
- 📱 **Responsive** — works on mobile and desktop

## 🚀 Quick Start

No build step needed. Just open `index.html` in a browser:

```bash
git clone https://github.com/YOUR_USERNAME/meme-machine-3000.git
cd meme-machine-3000
open index.html        # macOS
# or
xdg-open index.html   # Linux
# or just double-click index.html on Windows
```

### Or serve locally:
```bash
npx serve .
# Visit http://localhost:3000
```

## 📁 Project Structure

```
meme-machine-3000/
└── index.html     # The entire app — one file, no dependencies
```

## 🛠️ How to Use

1. **Pick a template** from the grid, or **upload your own image**
2. Type **Top Text** and/or **Bottom Text**
3. Customize **font, size, color, and stroke**
4. Add **stickers** (click an emoji to drop it on the canvas)
5. **Drag** text layers and stickers to reposition them
6. Hit **⬇ DOWNLOAD MEME** — done!
